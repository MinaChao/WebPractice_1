## 心得
介紹了自己喜歡的漫畫作品，試著讓網頁更完整，雖然有調整一些RWD，但用其他螢幕觀看排版有時候可能還是會跑掉。用Bootstrap切版很快速，但是class越加越多，結果程式碼感覺越打越亂😵，但還是順利完成了。下一次練習會再更好的判斷何時該加Bootstrap class。
